Jslet - 演示说明

Jslet秉承的原则是专业的人做专业的事，因此使用多个业界著名的免费开源的JavaScript组件包，具体如下：

  1. 基础组件：jQuery，DOM操作神器，不用对不住自己呀；
  2. 样式和布局: BootStrap，Jslet本身的样式均是以“jl-”开头，其它基本上是BootStrap的样式，大家遇到了可去查文档；
  3. 字体图标: FontAwesome, 样式以“fa-”开头；
  4. 图表得感谢百度的同学们: ECharts，这套图表功能非常强大，不过Jslet只选取了几种最常用的图表；
  5. Excel导入导出：Xlsx；
  6. 演示程序的模块加载: RequireJS

以上对于专业的前端人员可能根本不算事儿，早就烂熟于心了，但对于刚接触前端开发的人来说，还是有些难度，还是需要做些功课的，至少要知道谁是谁。

如果实在没有精力去了解也无妨，Jslet的演示程序本来就是基于标准的产品模式来做的，把演示程序改一改就可以立马发布了。

Jslet是基于现代的浏览器开发的，版本低的，比如IE10以下的就不要试了，目前浏览器还是以Chrome性能最佳，建议使用它。
